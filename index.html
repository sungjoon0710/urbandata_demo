This is a simple introduction to GitHub repositories and GitHub Pages.

What is GitHub?

GitHub is a platform for hosting and collaborating on code repositories.

What are GitHub Pages?

GitHub Pages is a service that allows you to host static websites directly from a GitHub repository.

Getting Started

  1. Create a new repository on GitHub.
  2. Add your HTML files to the repository.
  3. Enable GitHub Pages in the repository settings.
  4. Visit your new website at https://your-username.github.io/your-repository.

Happy coding!
